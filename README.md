# Assignment-5
 Dictionaries and Lists

Task 1: Create a Dictionary of Student Marks
This program creates a dictionary where student names are keys and their marks are values.  
It asks the user to enter a student's name and displays the corresponding marks.  
If the student name is not found in the dictionary, it displays an appropriate message.

Concepts Used:
- Dictionary
- Key-value pairs
- if-else condition

Task 2: Demonstrate List Slicing

This program creates a list of numbers from 1 to 10.  
It extracts the first five elements using list slicing and then reverses those elements.  
Finally, it prints the original list, extracted list, and reversed list.

Concepts Used:
- Lists
- range()
- List slicing
- Reverse slicing
